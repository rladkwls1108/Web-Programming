# 🚗 스마트 올팍 파킹

공연 관람객을 위한 올림픽공원 주변 최적 주차장 추천 서비스

## 📌 프로젝트 소개

올림픽공원에서 열리는 공연을 관람할 때, 출발지와 공연 시간을 입력하면 최적의 주차장과 출발 시간을 추천해주는 웹 서비스입니다.

## ✨ 주요 기능

- **실시간 공연 검색**: 한국문화정보원 API를 통한 공연 정보 조회
- **최적 주차장 추천**: 출발지 거리, 도보 거리, 주차 용량을 종합한 점수 기반 TOP 3 추천
- **권장 출발 시간 계산**: 차량 이동 + 주차 + 도보 시간을 고려한 출발 시간 안내
- **카카오맵 길찾기 연동**: 원클릭 네비게이션
- **QR 코드 생성**: 모바일로 길찾기 링크 공유

## 🛠️ 사용 기술

- HTML5 / CSS3 / Vanilla JavaScript
- Pretendard 웹폰트
- QRCode.js

## 📡 연동 API

| API | 제공처 | 용도 |
|-----|--------|------|
| 공연정보 API | 한국문화정보원 | 실시간 공연 목록 조회 |
| 운영현황 API | 한국문화정보원 | 공연장 운영 정보 |
| 공영주차장 API | 서울열린데이터광장 | 주차장 위치 및 요금 정보 |
| Kakao Maps | 카카오 | 길찾기 연동 |

## 🏟️ 지원 공연장

- 올림픽홀
- 체조경기장
- 펜싱경기장
- KSPO DOME

## 🚀 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 공연 선택 또는 공연장 직접 선택
3. 출발지, 공연 시간, 여유 시간 입력
4. "최적 주차장 찾기" 클릭
