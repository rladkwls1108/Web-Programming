# InterviewMate - 면접 연습 플랫폼

## 📝 프로젝트 소개

**InterviewMate**는 취업 준비생들을 위한 AI 기반 면접 연습 플랫폼입니다. 
다양한 분야와 난이도의 면접 질문을 통해 체계적으로 면접 능력을 향상시킬 수 있습니다.

## ✨ 주요 기능

### 1. 면접 연습
- **분야별 질문**: IT/컴퓨터, 경영/경제, 법학, 의학, 공학 등 11개 분야
- **난이도 선택**: 초급, 중급, 고급 3단계 선택 가능
- **카테고리**: 인성, 전공, 상황 면접 중 선택
- **랜덤 셔플**: 매번 다른 순서의 질문으로 학습

### 2. 답변 작성 및 저장
- 텍스트 기반 답변 작성
- 실시간 글자 수 카운팅
- 답변 자동 저장

### 3. 음성 녹음 및 인식
- 🎤 마이크로 음성 녹음
- 🔊 Web Speech API 기반 음성인식 (한국어 지원)
- 음성-텍스트 자동 변환
- 녹음 파일 저장

### 4. 피드백
- 🤖 작성한 답변에 대한 자동 분석
- 강점 및 개선점 제시
- 예상 채점 (0-100점)
- 꼬리 질문 제안

### 5. 타이머 기능
- ⏱️ 면접 시간 제한 연습
- 시작, 일시정지, 초기화 기능

### 6. 학습 히스토리
- 📚 최근 답변 자동 저장 (최대 50개)
- 질문, 답변, 작성 시간, 카테고리 기록
- 음성 파일 재생 가능

### 7. 통계 및 분석
- 📊 총 질문 수, 답변 수 통계
- 평균 작성 시간, 평균 글자 수
- 카테고리별 진행률 시각화
- 📄 리포트 다운로드 (TXT 형식)

### 8. 사용자 인증
- 회원가입 / 로그인
- 사용자별 독립적인 학습 데이터 관리

## 🎨 UI/UX 특징

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 대응
- **다크모드**: 🌙 테마 토글 기능
- **직관적 인터페이스**: 탭 기반 네비게이션
- **실시간 피드백**: 토스트 메시지로 사용자 액션 안내

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: LocalStorage (사용자 데이터 로컬 저장)
- **API**: Web Speech API (음성인식)

- **Media**: MediaRecorder API (음성 녹음)
